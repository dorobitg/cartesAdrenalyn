🔗 Compartir via GitHub Pages

Amb GitHub Pages pots allotjar l'app gratuïtament i accedir-hi des de qualsevol dispositiu amb una URL fixa. Només cal fer-ho una vegada.

PAS 1 — Crea un compte GitHub

Ves a github.com i registra't (és gratuït).

PAS 2 — Crea un repositori nou

Clica el botó + → New repository. Posa-li un nom (ex: adrenalyn), marca'l com a Public i clica Create repository.

PAS 3 — Puja el fitxer HTML

A la pàgina del repositori, clica uploading an existing file, arrossega el fitxer adrenalyn_col·leccio.html i canvia el nom a index.html. Clica Commit changes.

PAS 4 — Activa GitHub Pages

Ves a Settings → Pages. A Source selecciona Deploy from a branch, tria la branca main i clica Save.

PAS 5 — Accedeix a la URL

Al cap d'un minut la teva app estarà a:
https://[el-teu-usuari].github.io/adrenalyn/
Comparteix aquest link amb qui vulguis. Funciona en mòbil i ordinador.

💡 Per actualitzar l'estat: quan vulguis compartir els teus canvis, desa el fitxer .json i envia'l a l'altra persona perquè el carregui amb el botó 📂 Carregar.
